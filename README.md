## ‚ö° Installation

1. **Clone the repository or download ZIP**

```bash
git clone https://github.com/<your-username>/YT-MP3-Downloader.git

```

2. **Install Python**

* Python 3.10+ required
* Verify installation:

```bash
python --version
```

3. **Install dependencies**

```bash

```

4. **Install FFmpeg**

* **Windows:** Download FFmpeg ‚Üí extract ‚Üí add `bin` folder to PATH
* **macOS:**

```bash

```

* **Linux (Ubuntu/Debian):**

```bash
sudo apt install ffmpeg -y
```
